# Шаблон Python-сервиса

Шаблон даёт минимальный старт для микросервиса на Python.

## Содержимое
- `src/__init__.py` и `src/app.py` — точка входа; адаптируйте под CLI, worker или веб-приложение.
- `tests/test_smoke.py` — пример теста, использующий стандартный `pytest`.
- `Dockerfile` — многостадийная сборка с установкой зависимостей из `requirements.txt` (создайте файл при необходимости).
- `AGENTS.md` — локальные правила; дополните после настройки сервиса.

## После генерации
1. Создайте `requirements.txt`/`pyproject.toml` (или переиспользуйте общий менеджер зависимостей).
2. Обновите `Dockerfile`: добавьте установку зависимостей и корректную `CMD`.
3. Настройте pytest/ruff/mypy при необходимости и подключите сервис к общему `make tests`.
4. Обновите документацию сервиса (README + AGENTS).
