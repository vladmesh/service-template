# Шаблон сервиса (default)

Эта заготовка создаёт минимальную структуру для любого сервиса.

## Что входит
- `src/` — рабочий код сервиса. В шаблоне пустой каталог.
- `tests/` — будущие тесты; заполните минимальными smoke-проверками.
- `Dockerfile` — общий каркас с плейсхолдерами. Необходимо адаптировать под конкретный стек.
- `AGENTS.md` — локальные правила; обновите их после инициализации.

## Следующие шаги после генерации
1. Переместите или напишите код внутри `src/`.
2. Добавьте тесты и подключите их к общему `make tests` через конфигурацию сервиса.
3. Обновите Dockerfile: установите зависимости, настройте команду запуска, окружение.
4. Продублируйте важные инструкции в `README.md` и `AGENTS.md` сервиса.
