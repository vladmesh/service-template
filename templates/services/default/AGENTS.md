# AGENTS — шаблон сервиса (default)

- Этот сервис создаётся через `make sync-services create`; не удаляйте и не переименовывайте базовые каталоги `src/` и `tests/`.
- Любые команды запускайте через `make` и соответствующие цели (см. корневой `AGENTS.md`).
- Собственная автоматизация сервиса должна жить внутри контейнеров compose; не запускайте инструменты напрямую на хосте.
- После наполнения шаблона отредактируйте этот файл под конкретный сервис.
