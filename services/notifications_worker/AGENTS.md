# AGENTS — Python-сервис

- Код и зависимости держите внутри каталога сервиса; не выносите артефакты за пределы `services/<service>`.
- Все команды запускайте через `make` и подготовленные контейнеры (см. корневой `AGENTS.md`).
- Dockerfile уже содержит базовый образ `python:3.11-slim`; адаптируйте под нужные зависимости, не устанавливая их на хосте.
- После генерации допишите специфичные правила (используемый веб-фреймворк, способы запуска миграций и т.д.).
